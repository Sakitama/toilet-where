# toilet-where
人有三急，这款 App 是解决您燃眉之急的神器！
基于 react-native 开发，使用高德定位 api，自动搜索附近2km以内的公共厕所，解决您的燃眉之急！
本 App 还提供阅读功能，打发您的无聊时光！另外提供天气实时查询功能，让您的出行更加容易！
